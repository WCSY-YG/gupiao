# Report Summaries

This directory is for small, shareable Markdown summaries generated from local scans.

Do not commit raw market data, SQLite databases, per-symbol JSONL files, or full generated report directories. Keep heavy outputs in:

- `data/cache/`
- `reports/generated/`

Commit only compact summaries that are useful for reviewing scan parameters, aggregate counts, top candidates, failures, and risk notes.
